# 智慧調度系統 — OR 最佳化排程

單頁 HTML 應用程式,用於公共運具(如 YouBike)的**補車調度**規劃。
給定時間預算(例如 60 分鐘)和車速 / 壅塞係數,自動排出**最大補車數**的調度任務。

## ✨ 主要特色

- 🎯 **目標**:在時間預算內最大化總補車數
- 🔬 **演算法**:Beam Search 多路徑搜尋(避開貪婪陷阱)
- 🚗 **含銜接行車**:趟次之間的移動時間計入總預算
- ⏱️ **時間組成**:站內整備 + 裝卸 + 站間行車 + 趟次銜接
- 📍 **路網距離**:直線 × 1.4 道路係數
- 🎛️ **參數可調**:責任區、時段、車速、壅塞係數、預算分鐘、單趟最大運量

## 🚀 線上使用

## ⚙️ 關鍵參數(可在 `index.html` 內調整)

| 常數 | 位置 | 說明 |
|---|---|---|
| `SETUP = 5` | 約第 128 行 | 每站整備時間(分鐘) |
| `MIN_Q = 3` | 約第 128 行 | 單趟最低裝載門檻(輛) |
| `ROAD_K = 1.4` | 約第 128 行 | 路網 vs 直線距離係數 |
| `BEAM_WIDTH = 8` | 約第 263 行 | Beam Search 保留路徑數(越大越準但越慢) |
| `BRANCH_N = 6` | 約第 264 行 | 每個狀態展開的候選趟次數 |
